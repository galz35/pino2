7.3 DEPLOY PLAN
=================

a) Backend (VPS 190.56.16.85):
   1. Ejecutar migración SQL: psql -f migrations/2026-04-20_distribucion.sql
   2. npm run build en backend/
   3. Reiniciar servicio PM2: pm2 restart multitienda-api
   4. Verificar logs: pm2 logs multitienda-api --lines 50

b) React Web (Firebase Hosting):
   1. npm run build en web/
   2. firebase deploy --only hosting
   3. Verificar en https://pino-5fe44.web.app

c) Flutter Mobile:
   1. flutter build apk --release
   2. Distribuir APK vía Firebase App Distribution o link directo
   3. Verificar en dispositivos Android de preventas, bodegueros y ruteros

d) Git:
   1. git add . && git commit -m "feat: sistema de distribución bodega completo"
   2. git push origin main

===============================

  FASE 1 (BD) ──────┐
                     ├──→ FASE 2 (Backend) ──→ FASE 3 (React)
                     │                    └──→ FASE 4 (Flutter Preventa)
                     │                    └──→ FASE 5 (Flutter Bodeguero)
