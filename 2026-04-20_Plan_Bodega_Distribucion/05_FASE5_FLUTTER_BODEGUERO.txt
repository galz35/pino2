5.2 NUEVO: picking_checklist_screen.dart
==========================================
Archivo: flutter/lib/features/warehouse/presentation/screens/picking_checklist_screen.dart

Propósito: Lista de recolección (picking) con verificación obligatoria.

Flujo:
  1. Bodeguero selecciona un pedido en estado RECIBIDO o EN_PREPARACION
  2. Se muestra la lista de items con:
     ┌──────────────────────────────────┐
     │ □ Coca Cola 600ml                │
     │   ▸ 2 Bultos (24u c/u) + 6 unid │
     │   Pasillo: A3 - Estante: 2       │
     │   [____] Cantidad real alistada   │
     ├──────────────────────────────────┤
     │ □ Galleta María                  │
     │   ▸ 1 Bulto (12u c/u)           │
     │   Pasillo: B1 - Estante: 5       │
     │   [____] Cantidad real alistada   │
     └──────────────────────────────────┘
  3. El bodeguero DEBE marcar cada item como ✅ alistado
  4. Si la cantidad real difiere de la solicitada:
     - Campo de observación obligatorio
     - El sistema registra como "envío parcial"
  5. Solo cuando TODOS los items están checkeados → Botón "Confirmar Alistado"
  6. PATCH /api/orders/:id/status { status: 'ALISTADO' }

Consolidación matemática (se calcula localmente):
  - Para cada producto:
    * totalUnidades = cantidad del pedido
    * Si presentación = BULTO: totalUnidades = cantidad * unidadesPorBulto
    * bultos = totalUnidades / unidadesPorBulto (entero)
    * sueltas = totalUnidades % unidadesPorBulto (residuo)
  - Mostrar: "2 Bultos (24u c/u) + 6 unidades sueltas"

======================================
