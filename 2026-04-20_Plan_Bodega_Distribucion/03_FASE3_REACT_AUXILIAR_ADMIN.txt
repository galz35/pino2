3.1 PÁGINA: Panel de Despacho (Auxiliar)
==========================================
Ubicación: web/src/pages/store-admin/dispatch/dispatch-page.tsx
Ruta: /store/:storeId/dispatch

Funcionalidad:
  - Vista de pedidos por estado (columnas Kanban):
    * RECIBIDO → EN_PREPARACION → ALISTADO → CARGADO_CAMION → EN_RUTA
  - Arrastrar pedidos entre columnas (o botones de acción)
  - Cada tarjeta muestra: #pedido, cliente, total, items, peso estimado

Sub-funcionalidades:
  a) Asignación Masiva:
     - Seleccionar múltiples pedidos (checkbox)
     - Botón "Asignar a Rutero"
     - Modal: Seleccionar rutero + placa de camión + fecha entrega
     - POST /api/despacho/asignar-masivo
     
  b) Vista de Carga por Camión:
     - Agrupar pedidos por rutero/camión
     - Mostrar consolidación: lista de bultos + lista de unidades sueltas
     - Botón "Imprimir Hoja de Carga" (PDF)

3.2 PÁGINA: Autorización de Precios
======================================
Ubicación: web/src/pages/store-admin/authorizations/price-auth-page.tsx
Ruta: /store/:storeId/authorizations/prices

Funcionalidad:
  - Listar pedidos con status PENDIENTE_AUTORIZACION
  - Para cada pedido mostrar:
    * Cliente, preventa que lo levantó
    * Productos con precio solicitado vs precio estándar
    * Nivel de precio aplicado (4 o 5)
    * Diferencia en % respecto al precio normal
  - Botones: "Aprobar" | "Rechazar" con campo de observación
  - Al aprobar: POST /api/orders/:id/autorizar { decision: 'aprobar' }
  - Al rechazar: POST /api/orders/:id/autorizar { decision: 'rechazar', motivo: '...' }
