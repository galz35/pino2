# Gemini Import

Este directorio guarda el material descargado desde Gemini que todavía no forma parte de la app Flutter activa.

## Estructura

- `diagnostics/`
  - logs y reportes de análisis o compilación
- `source/`
  - código y artefactos experimentales importados

## Importante

Nada de este directorio se considera integrado al flujo activo de la app móvil.

La app activa sigue en:

- `../flutter/lib/app`
- `../flutter/lib/core`
- `../flutter/lib/features`

Este material queda como referencia para rescatar piezas útiles después, de forma controlada.
